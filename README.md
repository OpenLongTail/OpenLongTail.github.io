# OpenLongTail — Project Page

Project page for **OpenLongTail: Generative Scaling of Long-Tail Driving Data**.

Static site (HTML/CSS/JS) served via GitHub Pages.

- `index.html` — the page (generated by `build_html.py`)
- `assets/` — videos, posters, figures, css, js
- `.nojekyll` — disables Jekyll so all asset paths are served as-is

## Local preview
```bash
python3 -m http.server 8000
# open http://localhost:8000
```
